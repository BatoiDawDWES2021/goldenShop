 per funcionar: 
  * Obrir terminal i executar 
    > sh start.sh
  * Per executar composer, amb màquina encesa executar
    > sh composer.sh
  * Per finalitzar, executar
    > sh stop.sh 
  * Per executar test, amb màquina apagada executar
    > sh test.sh
